    • hookman 是基于github上的webhooks开发的一个用Python写的小程序
    ,基于TDD开发.
    如果你想向项目贡献代码,请看hookman 开发blog

    简单用法

    1. 安装 pip install hookman
    2. 配置github的webhooks,设置监听服务器ip和端口3610(默认)
    3. 使用
      3.1 进入github项目目录cd /my/github/projectdir
      3.2 运行 hookman --run -d
    4. 关闭
      3.1 hookman --stop
